# Health Prediction Application

## Project Overview
This application predicts possible health risks using patient blood test values and healthcare-related logic.

## Features
- Create, Read, Update, Delete (CRUD) operations
- Input validation
- SQLite database integration
- Healthcare prediction logic
- External FDA API integration
- Search functionality
- User-friendly Streamlit interface

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Requests API

## How to Run

Install required libraries:

pip install -r requirements.txt

Run application:

streamlit run app.py
